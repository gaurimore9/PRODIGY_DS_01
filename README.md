# Histogram Visualization using Real Dataset

## Objective
To visualize the distribution of a continuous variable using a real-world dataset.

## Dataset
- Breast Cancer Wisconsin (Diagnostic) Dataset
- Source: UCI Machine Learning Repository
- Loaded using scikit-learn

## Tools Used
- Python
- Matplotlib
- Scikit-learn

## Description
A histogram is used to show the distribution of the 'mean radius' feature from the dataset. 
This helps in understanding the spread and frequency of values in the data.

## How to Run
1. Install required libraries:
   pip install matplotlib scikit-learn
2. Run the script:
   python histogram_visualization.py

## Output
The output is a histogram showing the distribution of mean radius values.
